= 快速啟動設定 AFX 儲存系統
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
若要初始啟動並執行您的 AFX 系統，您需要安裝硬體元件、設定叢集，並準備管理叢集和 SVM。

.image:https://raw.githubusercontent.com/NetAppDocs/common/main/media/number-1.png["一"] AI Data Engine（AIDE）是否也會隨 AFX 一起安裝？
[role="quick-margin-para"]
您需要決定是僅安裝 AFX 還是同時安裝 AFX 和 AI Data Engine。請查看 https://docs.netapp.com/us-en/ontap-afx/get-started/learn-about-aide.html#installation-options["AIDE 安裝選項"]以協助您確定最適合您需求的方案。

.image:https://raw.githubusercontent.com/NetAppDocs/common/main/media/number-2.png["二"] 安裝並設定您的硬體
[role="quick-margin-para"]
link:../install-setup/install-setup-workflow.html["安裝您的 AFX 儲存系統"] 以及選擇性地 https://docs.netapp.com/us-en/ai-data-engine/install-setup/aide-hardware-requirements.html["安裝 AIDE 資料運算節點"]如果您將 AIDE 作為初始配置的一部分。

.image:https://raw.githubusercontent.com/NetAppDocs/common/main/media/number-3.png["三"] 設定叢集
[role="quick-margin-para"]
按照簡單快速的步驟使用系統管理器link:../install-setup/cluster-setup.html["設定"]您的 ONTAP 叢集。如果您也要在資料運算節點上安裝 AIDE 和 AFX，請查看 https://docs.netapp.com/us-en/ai-data-engine/install-setup/cluster-setup-afx.html["使用新的 AFX 叢集設定 AIDE"^]以瞭解其他必要的組態。

.image:https://raw.githubusercontent.com/NetAppDocs/common/main/media/number-4.png["四"] 準備管理您的叢集
[role="quick-margin-para"]
在生產環境部署 AFX 之前，必須link:../get-started/prepare-cluster-admin.html["準備"]透過了解管理結構（包括儲存虛擬機器 (SVM)、使用者、角色和管理介面），確保安全、高效和有效的叢集管理。
