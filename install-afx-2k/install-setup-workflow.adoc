= AFX 2K 儲存系統的安裝與設定工作流程
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
要安裝和設定 AFX 2K 儲存系統，您需要查看硬體需求、準備站台、安裝交換器、安裝和連接硬體元件、啟動系統並設定 ONTAP AFX 叢集。

.image:https://raw.githubusercontent.com/NetAppDocs/common/main/media/number-1.png["一"]link:../install-setup/install-setup-requirements.html["查看硬體安裝需求"]
[role="quick-margin-para"]
查看安裝 AFX 2K 儲存系統的硬體需求。

.image:https://raw.githubusercontent.com/NetAppDocs/common/main/media/number-2.png["二"]link:../install-setup/prepare-hardware.html["準備安裝您的 AFX 2K 儲存系統"]
[role="quick-margin-para"]
安裝 AFX 2K 儲存系統前，請先準備場地、檢查環境和電氣需求、確保足夠的機架空間、拆箱設備、核對裝箱單上的內容，並註冊硬體以獲得支援。

.image:https://raw.githubusercontent.com/NetAppDocs/common/main/media/number-3.png["三"]link:install-switches.html["為您的 AFX 2K 儲存系統安裝交換器"]
[role="quick-margin-para"]
在機櫃或機架中安裝 Cisco Nexus 9808、9332D-GX2B 或 9364D-GX2A 交換器。如果使用 Cisco Nexus 9364D-GX2A 交換器，請安裝直通面板套件。

.image:https://raw.githubusercontent.com/NetAppDocs/common/main/media/number-4.png["四"]link:deploy-hardware.html["安裝 AFX 2K 儲存系統的硬體"]
[role="quick-margin-para"]
安裝儲存系統和架子的導軌套件。將儲存系統固定在機櫃或電信機架中。接下來，將架子滑到安裝好的導軌上。最後，將電纜管理設備連接到儲存系統的後部，以便有組織地佈線。

.image:https://raw.githubusercontent.com/NetAppDocs/common/main/media/number-5.png["五"]link:cable-hardware.html["為您的 AFX 2K 儲存系統連接控制器和磁碟櫃"]
[role="quick-margin-para"]
要連接硬件，首先將儲存控制器節點連接到網絡，然後將控制器節點和儲存架連接到叢集交換器。

.image:https://raw.githubusercontent.com/NetAppDocs/common/main/media/number-6.png["六"]link:power-on-configure-switch.html["開啟 AFX 2K 儲存系統的電源並設定交換器"]
[role="quick-margin-para"]
啟動並設定 AFX 2K 儲存系統的交換器。請查閱 Cisco Nexus 9808、9332D-GX2B 和 9364D-GX2A 交換器的設定說明。

.image:https://raw.githubusercontent.com/NetAppDocs/common/main/media/number-7.png["七"]link:power-on-hardware.html["啟動您的 AFX 2K 儲存系統"]
[role="quick-margin-para"]
在開啟控制器節點的電源之前，請開啟每個儲存架的電源並指派唯一的儲存架 ID，以清楚辨識設定中的每個儲存架。
