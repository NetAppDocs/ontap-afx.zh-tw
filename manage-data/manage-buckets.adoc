= 管理 AFX 儲存系統儲存桶
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
在管理 AFX S3 儲存桶和用戶端存取時，您可以執行多項管理任務。 AFX 中的 S3 設定和支援與 Unified ONTAP提供的相同。有關詳細信息，請參閱 Unified ONTAP文件。

隨時了解 ONTAP 對 S3 儲存支援的最新變更和更新也至關重要。例如，從 ONTAP 9.19.1 開始，您可以為每個 S3 使用者產生第二個存取金鑰。這樣，即使原始存取金鑰過期，您仍然可以存取 S3 資源。您也可以控制條件操作（寫入、複製、刪除）的強制執行。
