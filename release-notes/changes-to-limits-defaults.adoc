[[changes-to-ontap-limits]]
== ONTAP 限制變更

[cols="1,3,1"]
|===
| 功能 | 限制變更 | 已在版本中變更 


| 每個叢集的節點數  a| 
您可以將 AFX 叢集擴充到最多 32 個節點。這樣，您可以擴充 AFX 儲存系統，以滿足不斷增長的工作負載和應用程式需求。請注意，32 個節點的支援功能有限；詳情請諮詢您的 NetApp 帳戶經理。
| ONTAP 9.19.1 


| 每個 FlexGroup 的 Volume 數  a| 
FlexGroup 中的最大 Volume 數量已增加至 512 個，節點數為 32 個。這使得 AFX 儲存系統能夠充分發揮其效能功能，以支援更大規模的叢集。
| ONTAP 9.19.1 
|===
