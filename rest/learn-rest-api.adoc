:linkattrs: 
:imagesdir: ../media/


[role="lead"]
AFX 提供的 REST API 是基於 Unified ONTAP REST API。有許多變化可以使其適應 AFX 個性的獨特特徵和能力。



== 不支援的功能

AFX 是一款高效能 NAS 和 S3 儲存系統。它使客戶端能夠使用 NFS、SMB/CIFS 和 S3 存取資料。由於這種專業化，有幾個不支援的功能，包括：

* 大都會集群
* 儲存區域網路 (SAN)
* 磁碟聚合
